### Software 3.0

> [Karpathy's Software 3.0](https://mp.weixin.qq.com/s/WNbnl1ORlqgPxWwwM399ew)

> [Vibe Vibe (Community Resource)](https://github.com/datawhalechina/vibe-vibe)

The emerging paradigm of Software 3.0 reframes how software is built and what engineers do:

- Software 1.0: Hand-written code, explicit logic
- Software 2.0: Neural networks trained on data (weights = program)
- Software 3.0: LLMs as programmable reasoning engines, instructed via natural language
