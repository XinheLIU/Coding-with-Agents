# Reframing Code Review

Traditional code reviews are overwhelmed at AI volume. The new goal is to keep the team aligned on *what's changing and why*.

- ✅ Read 200 lines of a good plan
- ❌ Skip 2,000 lines of generated code

Shift alignment to specs, research docs, and plans — instead of raw diffs.

## AI-Assisted Support, Debugging & Code Review

> [Graphite — AI-Native Code Review Platform](https://graphite.dev)

**Graphite** is an AI-native platform designed to help teams create, review, and merge code faster. Key capabilities:
- **AI code review:** explains PRs, proposes updates, and proactively scans for bugs, security vulnerabilities, and logic errors
- **Stacking:** a git workflow that lets authors keep developing without being blocked waiting for review
- **Smarter CI:** predictive CI that runs only when necessary to save time and resources
- **Insights:** measures developer productivity and velocity (e.g., time between PR merges)

- AI-powered code review
- Debugging with reasoning traces
- Generating and maintaining documentation
